# Roadmap: Resume Builder MVP

Цель: Создать веб-приложение для генерации резюме с фокусом на шаблон "Classic".

## Фаза 1: Инициализация и Архитектура (Sprint 1)
- [ ] **Настройка проекта:** Инициализация Next.js + TS + Tailwind.
- [ ] **Настройка шрифтов:** Подключение `Merriweather` и `Open Sans`.
- [ ] **Определение структуры данных:** Создание TypeScript интерфейсов (`types.ts`) для объекта Resume (PersonalDetails, Experience, Education, Skills).
- [ ] **Настройка Store:** Настройка Zustand стора для хранения и обновления данных резюме.

## Фаза 2: UI Редактора (Sprint 2)
- [ ] **Layout приложения:** Разделение экрана на 2 части (Editor - 40%, Preview - 60%).
- [ ] **Компонент формы "Личные данные":** Поля ввода (Имя, Должность, Email, Телефон). Связка с Store.
- [ ] **Компонент формы "Опыт работы":**
    - [ ] Форма добавления одного места работы.
    - [ ] Список добавленных мест.
    - [ ] Функционал удаления/редактирования.
- [ ] **Компоненты форм "Образование" и "Навыки".**

## Фаза 3: Разработка Шаблона "Classic" (Sprint 3)
*Фокус на типографике и отступах.*
- [ ] **Контейнер А4:** Верстка div-блока с пропорциями 210mm ширина (высота auto).
- [ ] **Header шаблона:** Верстка имени и контактов (строгий стиль, центрирование или левый край).
- [ ] **Секции:** Верстка блоков с опытом и образованием. Использование тегов H2, H3, UL, LI.
- [ ] **Стилизация:** Применение шрифтов с засечками, черно-белая гамма, линии-разделители.
- [ ] **Проверка переполнения:** Тест на длинных текстах.

## Фаза 4: Логика Печати и PDF (Sprint 4)
- [ ] **CSS Print Media:** Написание стилей `@media print`.
    - [ ] Скрытие панели редактора.
    - [ ] Установка `page-break-inside: avoid` для блоков опыта (чтобы текст не рвался посередине абзаца).
- [ ] **Интеграция react-to-print:** Добавление кнопки "Скачать PDF", вызывающей нативное окно печати.

## Фаза 5: Оптимизация и Полировка (Sprint 5)
- [ ] **Live Preview:** Обеспечение мгновенного обновления шаблона при вводе букв в редакторе.
- [ ] **Local Storage:** Реализация сохранения прогресса (если пользователь закрыл вкладку, данные не должны исчезнуть).
- [ ] **Mobile Responsive:** Адаптация панели редактора для мобильных устройств (превью скрывается или уходит вниз).

## Фаза 6: Масштабирование (Post-MVP)
- [ ] Добавление шаблона "Modern".
- [ ] Добавление шаблона "Creative".
- [ ] Добавление выбора цвета акцентов.
